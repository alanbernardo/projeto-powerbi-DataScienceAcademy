# Dashboard Analítico de Vendas Globais

Projeto de dashboard desenvolvido em Power BI Desktop como parte das atividades práticas realizadas em conjunto com a Data Science Academy, com foco em análise exploratória e visualização de dados de vendas globais.

## Objetivo
Apresentar indicadores e análises de vendas por categoria, subcategoria, país e período, permitindo uma visão consolidada do desempenho comercial por meio de filtros e visualizações interativas.

## Principais Análises
- Total de vendas por categoria
- Média de desconto por subcategoria
- Distribuição de vendas por país
- Segmentação por ano e segmento de clientes

## Ferramentas e Tecnologias
- Power BI Desktop  
- Power Query (ETL e transformação de dados)  
- Visualizações interativas

## Observações
- O projeto tem finalidade educacional e prática.
- Dependendo da configuração de dados locais, o arquivo `.pbix` pode abrir sem permitir atualização automática do dataset.
